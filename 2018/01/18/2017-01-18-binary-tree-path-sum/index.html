二叉树的路径和

http://www.lintcode.com/zh-cn/problem/binary-tree-path-sum/
链接

题目内容

给定一个二叉树，找出所有路径中各节点相加总和等于给定 目标值 的路径。
一个有效的路径，指的是从根节点到叶节点的路径。

分析

上来第一个想法，把所有路径和对应的和都计算出来，放在一个map中。计算完成之后遍历map，找到其中和符合结果的，把路径放到最后的结果中。

看书得到的想法

通常来说关于树的算法题难免要使用递归。计算路径和可以认为是一个遍历树的过程。在前序遍历中，最先出现是根节点符合路径的要求。从根节点依次向下去遍历。把子节点的值加入到路径中，最后在叶子节点上停止。若符合要求则把该路径加入结果集合。若不符合则回退到父节点，再从父节点访问其他子节点的路径。
值得注意的是在回退之前需要把最后节点从队列中去除。

实现

算法入口

首先是调用函数，也是算法的入口。
需要注意空指针
在一开始没考虑传入 root 为 null 的情况。
在运行算法的时候发现不符合测试用例。后来发现测试用例传入一个空树，但是要求返回一个空数组。这就提醒我们需要注意边界条件。不仅仅是返回一个 null 还要注意题目的要求。

递归函数

按照之前的思路，递归过程首先是分为四部分

  • 把当前节点加入路径 & 更新路径和
  • 若当前节点是子节点 & 路径和符合结果 => 输出结果到集合中
  • 若不符合结果 => 对子节点进行递归
  • 在路径中去除当前节点

遇到的问题

最重要的问题就是在输出结果的时候,若直接把之前的传入参数 path 保存到数组中,会出行问题.主要原因就是 Java 的数组是直接保存传入的对象引用.若该对象发生改变数组中的内容也会改变.也就是说直接传 path 进去,最后的结果不是我们想要的值.
正确的方式是产生一个 path 的拷贝.作为结果传入.
