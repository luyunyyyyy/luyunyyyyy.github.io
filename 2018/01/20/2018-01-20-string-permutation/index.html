发表于 2018-01-20
字数统计: 352 字 | 阅读时长 ≈ 1 分钟

题目内容

给出的示例

如 abc 是 cba 的置换
aabc 不是 abcc 的置换

边界条件

首先要考虑输入数据的边界情况。

  • 字符串有 null
  • 字符串为 ""
  • 字符串大小不相等

思路

二逼青年思路

把两个字符串分别保存成两个 HashMap 存储每个字符的出现次数。
这就需要把两个字符数组变成 HashMap。然后比较两个 Map

普通青年思路

对两个字符数据进行排序，若排序结果相同则表示两个字符串中字符的出现频率相同。

文艺青年思路

还是使用 Map
但是在每次遍历的时候都同时操作两个字符串。若 A 中存在则该字符加一，若 B 中存在则减一。同时保持 Map 中 value 为0的不存在，去除掉。即最后只要判断 Map 大小是否为0.
