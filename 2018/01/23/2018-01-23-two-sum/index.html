暴力搜索

使用一个二重循环进行遍历，判断是否有符合条件的数。符合就返回一个数组。
这里遇到一个小问题，Java 中如何创建数组。

Java 中创建数组的几种方式

  1. type[] arrayRefVar; 或 type arrayRefVar[]; arrayRefVar = new dataType[arraySize];
  2. arrayName = new type[]{element1,element2,element3…}
  3. 或者使用简化的语法格式: arrayName = {element1,element2,element3…}
